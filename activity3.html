ACTIVITEIT VERSLAG

Selux riolerings applicatie

MEAN-stack

Een kennis van mij zocht iemand die mee kon helpen met het verder bouwen van een rioleringsapplicatie (Selux) die gebouwd is met de MEAN-stack (MongoDb, Express, Angular, NodeJs). Omdat ik al met Angular ervaring had en mij hier nog verder in wou verdiepen heb ik de kans gegrepen en deze uitdaging aangegaan.

Selux is dus een rioleringsapplicatie dat rioleringsbedrijven kunnen gebruiken om projecten te digitaliseren. Hierbij kunnen ze verschillende projecten aanmaken, foto’s toevoegen, de nodige materialen ingeven per project, Excel en/of pdf-documenten laten genereren en nog meer.

Mijn taken waren divers in dit project. Ik heb zowel front-end als back-end deze kunnen uitvoeren. Ik heb verder gewerkt op de admin project van Selux en op de mobiele versie met Ionic.
Hierin heb ik verschillende features moeten implementeren. Zoals, voorzien om bepaalde soorten projecten (DWA, RWA, slokkerprojecten) te kunnen aanmaken. Elk soort project heeft zijn eigen datastructuur waarbij deze data ook opgeslaan, bewerkt of verwijderd kan worden. Ik moest hiervoor dus de nodige pagina met Angular aanmaken en uiteindelijk opslaan in onze databank.

Een ander voorbeeld van mijn implementaties was een tabel met verschillende filter functies voorzien om projecten te kunnen visualiseren en selecteren.

Ook moest er een settingspagina voorzien worden waarbij elke gebruiker bepaalde configuraties kon aanvinken zoals variabelen aan en uit zetten.

Een mobiele versie (Ionic) was ook voorzien, waarbij werknemers bijvoorbeeld gegevens per project konden aanpassen en foto's toevoegen.

Hierbij ben ik wel wat problemen tegengekomen. Omdat er bij het aanmaken/bewerken van een project sequentieel formulieren moest ingevuld worden heeft dit wat voor problemen gezorgd. Het effectief uploaden en weergeven van foto’s was ook een leuke uitdaging. Uiteindelijk door zelf te experimenteren en oplossingen te zoeken was het ons gelukt om dit probleem aan te pakken.

Dit was in het algemeen een heel leuk uitdaging omdat ik redelijk weinig ervaring had met NodeJs en had hiermee ook de kans om hierin te werken.

Voor het Angular deel was het ook heel interessant om mijn frontend skills te verbeteren. Na elk afronding van een feature geeft dit ook een voldoening en motiveert dit jou om er iets leuk van te maken.

Doorheen het project ben ik uiteraard veel problemen tegengekomen en heb geprobeerd om van elk probleem mijn les uit te trekken. Voor het backend deel heb ik mij in het begin ook extra moeten inzetten omdat ik hier redelijk weinig ervaring in had.

Al bij al verliep deze proces wel heel goed. Ik heb reverse-engineering kunnen doen om de bestaande code te bestuderen en dus de algemene werking van de backend te begrijpen en hierna mijn eigen implementaties te kunnen inbrengen. Uiteindelijk verliep dit project zeer goed en was heel tevreden met de resultaten die geboekt waren. Ik heb zowel mijn front-end skills als backend skills kunnen verbeteren. Voor mijn studies heb ik dit ook zeker meegenomen.

Mijn uiteindelijke doel is als fullstack developer te kunnen werken en dit project was hierdoor een heel goede start.
